Ejercicio 8: Mayúsculas o minúsculas

Vamos a crear una página web en la que podamos introducir, a través de un cuadro de diálogo (prompt()), la frase que queramos y ésta nos dirá si está escrita en mayúsculas, minúsculas o ambas. Para ello, utilizaremos condicionales, funciones básicas para strings y operadores comparativos.
