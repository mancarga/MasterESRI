Ejercicio 3 Operaciones con Arrays

A partir de los datos del siguiente array realiza las siguientes acciones y muestra los resultados por la consola:

const data = [“hola”, 2, 5, “adios”];

  1. Determina cuál de los dos números que hay en el array es mayor.
  2. Determina el resultado de las cinco operaciones numéricas realizadas con los dos elementos numéricos.
